# intro
This is a test post. I came to the conclusion that the posts (weather they belong to blog or simulations) is better to be in the following format: **Page - Article Header Overlay Background Image (Customized Header Background)**

The reason is:
- one can uses the cover image as the grid image if the archive is in grid mode.
- the costumized nav bar on top disticts the post from the home page where the image immerses the navigation.
